# Salesforce CRM in Planit’s Operational Success

IT strategy report on how Planit can leverage **Salesforce CRM** to improve customer relationship management and streamline operations.

## Overview
- **Context:** Planit is a global quality engineering & assurance company.
- **Goal:** Evaluate how Salesforce CRM can improve service delivery, customer satisfaction, and operational efficiency.
- **Scope:** Benefits, deployment risks (inhibitors), success factors (enablers), and organisational impact.

## Key Takeaways
### Expected Benefits
- Centralised customer information for faster decision-making and response.
- Process automation (sales/support/reporting) to reduce repetitive work and improve efficiency.
- Better customer engagement through data-driven and AI-assisted workflows.

### Deployment Risks (Inhibitors)
- High initial setup and licensing cost
- Data migration complexity
- User adoption / change resistance
- Integration effort with existing tools and systems

### Success Factors (Enablers)
- Strong leadership & clear rollout plan
- Role-based, practical user training
- Customisation aligned to internal workflows
- Integration testing and phased implementation (pilot → expand)
